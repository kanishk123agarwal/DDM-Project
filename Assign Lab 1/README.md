# DDM Lab Exercises

This repository contains Python scripts implementing various data processing and programming tasks as defined in `instruction.txt`. The exercises cover data parsing, binary file operations, regular expressions, SQL and MongoDB databases, and data analysis using NumPy and Pandas.

## Prerequisites

Ensure you have Python installed. The following external libraries are required for some scripts:

```bash
pip install numpy pandas matplotlib pymongo
```

*Note: `pymongo` is required for `05_mongodb.py`. If you don't have a MongoDB server running, the script will skip database operations gracefully.*

## Project Structure

### Data Files
- **`patient_consent.csv`**: The primary dataset used for parsing and database population.
- **`instruction.txt`**: The original problem set description.
- **`sample.txt`, `sample.html`, `sample.xml`, `sample.json`**: Dummy files generated by `01_data_parsing.py` for demonstration.

### Python Scripts

| Script | Description |
| :--- | :--- |
| **`01_data_parsing.py`** | Parses Text, CSV, HTML, XML, and JSON files. Checks for anomalies (missing values) in the CSV data. |
| **`02_binary_files.py`** | Demonstrates reading and writing binary files using Python's `pickle` and `struct` modules. |
| **`03_regex.py`** | Performs string searching, splitting, and replacing using regular expressions (e.g., extracting emails, redacting phone numbers). |
| **`04_sql_db.py`** | Creates a SQLite database (`medical_study.db`), populates it with CSV data, and performs CRUD (Create, Read, Update, Delete) operations. |
| **`05_mongodb.py`** | Connects to a local MongoDB instance to perform CRUD operations and data aggregation. Handles connection failures if MongoDB is not running. |
| **`06_numpy_ops.py`** | Demonstrates NumPy array creation, reshaping, slicing, arithmetic, and aggregation functions. |
| **`07_pandas_ops.py`** | Uses Pandas for data analysis: handling missing data, calculating new columns, grouping/aggregation, and plotting results. |

## How to Run

Run any script using the python command:

```bash
python 01_data_parsing.py
python 07_pandas_ops.py
# etc...
```

## Outputs

- **`medical_study.db`**: SQLite database file created by `04_sql_db.py`.
- **`data.pickle`, `data.bin`**: Binary files created by `02_binary_files.py`.
- **`study_scores_plot.png`**: Bar chart generated by `07_pandas_ops.py` visualizing average scores by study ID.
